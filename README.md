# Task Manager Backend

This is the backend of the Task Management Web App. It is built with Node.js, Express.js, and MongoDB (Mongoose) to provide a RESTful API for tasks and user authentication.

---

## **Tech Stack**

- **Node.js** – JavaScript runtime for backend
- **Express.js** – Web framework for API routing
- **MongoDB Atlas** – Cloud database for storing users and tasks
- **Mongoose** – ODM for MongoDB
- **JWT (jsonwebtoken)** – Authentication and authorization
- **bcryptjs** – Password hashing

---

## **Features**

- User registration and login (JWT-based)
- Create, read, update, delete tasks
- Mark tasks as completed or pending
- Task ownership enforced per user
